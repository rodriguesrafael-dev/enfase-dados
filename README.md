# Enfase-Dados (Projeto de teste de seleção para desenvolvedor full stack Java) - ESIG Group Project

# Tecnologias usadas
## Backend
* Spring Boot 2.6.2 
* Java 11
* PostgreSQL 42.2.20
* Auth0 3.16.0
* Lombok
* Maven
* Swagger 3.0.0

## Frontend
* Angular 11
* TypeScript

# Caracteristicas
* Validação de usuário, com token e senha encriptografada
* Documentação de API

# Executar projeto
## Server side
* porta 8080
* mvn springboot:run

## Client side
* porta 4200
* ng serve
